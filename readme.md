PSmacOS
========

PowerShell module containing convience or platform-specific cmdlets for macOS.

## Cmdlet list

`Get-Clipboard` - get clipboard content as a string and return to the pipeline

`Set-Clipboard` - set clipboard content

## Planned/Future
